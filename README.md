# Python-Venn-Diagram-and-Set-Display
A program created to display the overlap between upwards of three sets of elements via a dynamic venn diagram display.
Users can open the program either by running the Venn_GUI_Main.py file or by using the code python -m Venn_GUI_Main from the command line directly.
Elements can consist of any alphanumeric characters, and can be multiple words in length. Any unaccepted inputs will be listed as error through the command line.
Users can create and save any number of sets per file, and can display and compare up to any three at a time.
Sets can be created, deleted, and modified. The number of sets and unique elements are tracked for the user.
Try loading the set venntest as a demo!
